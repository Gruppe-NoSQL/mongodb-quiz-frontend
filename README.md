# nosql-quiz

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### build docker container
```
    docker build -t frontend .
```

### run docker container
```
    docker run -it -p 8080:80 frontend
```
